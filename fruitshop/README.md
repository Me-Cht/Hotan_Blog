# Hotan-Blog个人博客平台

## 项目简介

该项目是一个个人独立开发的博客平台，它使用了Vue和elementUI作为前端技术栈，并且后端采用了Spring Boot框架来创建。

### **下载所需的包到项目中**

```
npm install
```

### 运行前端项目
```
npm run serve
```

### 进行生产环境的编译和代码压缩
```node.js
npm run build
```

[项目后端接口GitHub源码支持](https://github.com/Me-Cht/Hotan-Blog-SpringBoot.git)

### 本项目已实现的部分功能：

- 登陆功能
- 注册功能
- 博客发表功能
- 个人中心查看个人博客及个人信息
